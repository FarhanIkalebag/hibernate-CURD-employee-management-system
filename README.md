# Hibernate Employee Management System

A console-based Employee Management System built using **Java** and **Hibernate ORM**, demonstrating complete **CRUD (Create, Read, Update, Delete)** operations with a **MySQL** database.  
This project focuses on core Hibernate concepts such as entity mapping, session handling, and transaction management.
